# Billing Software in Python 🧾

This is a GUI-based Billing Software developed using Python and Tkinter.

## 🚀 Features
- Add customer details
- Select products (Cosmetics, Grocery, Cold Drinks)
- Automatic total and tax calculation
- Generate bill
- Save bill as text file

## 🛠️ Technologies Used
- Python
- Tkinter (GUI)

## 📂 File
- billing_app.py

## ▶️ How to Run
1. Install Python
2. Run the file:
   python billing_app.py
